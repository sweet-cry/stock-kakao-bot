# 주식 분석 카카오톡 챗봇

## 사용 가능한 명령어
- `LCID 실적` → Q4 실적 분석
- `TSLA 유사기업` → Comps 분석
- `LCID 밸류` → DCF 목표주가
- `시장 브리핑` → 오늘 시장 요약
- `도움말` → 명령어 목록

## 배포 (Railway)
1. GitHub에 이 폴더 push
2. Railway에서 GitHub repo 연결
3. 자동 배포 완료 → URL 복사
4. 카카오 오픈빌더 스킬 URL에 입력

## 웹훅 URL
`https://[your-railway-url]/kakao`

## 헬스체크
`https://[your-railway-url]/health`
